- **官网：<https://serein.cc/>**
- **下载最新版：[Releases](https://github.com/Shiroiame-Kusu/Serein/releases/latest)**

## 建议运行环境

Win7或WinServer2012以上

> 更低的系统版本不保证能稳定运行，必要时务必备份数据
